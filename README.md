# animay 🌸

### A Modern Anime Discovery Experience

animay is a stylish anime discovery web app inspired by modern streaming platforms and Japanese UI aesthetics.
It combines smooth animations, immersive visuals, trending anime sections, dynamic search, and responsive layouts into a clean cinematic experience.

Designed with a futuristic anime-inspired interface, animay focuses on both aesthetics and usability while remaining lightweight and beginner-friendly.

---

# ✨ Features

* 🎌 Beautiful Japanese-inspired UI
* 🔥 Trending anime showcase section
* 🔎 Anime search interface
* 🎨 Smooth hover animations & transitions
* 📱 Fully responsive design
* 🌙 Cinematic dark mode aesthetics
* ⚡ Lightweight and fast
* 🧩 Modular and easy-to-edit structure
* 💖 Modern glassmorphism card design

---

# 🛠️ Built With

* HTML5
* CSS3
* JavaScript
* Google Fonts
* Responsive Flexbox/Grid Layouts
